# 📚 Arrays - Notes

## What is an Array?

An array is a collection of elements stored in a sequential manner.

Example:

```python
numbers = [10, 20, 30, 40, 50]
```

---

## Characteristics

- Stores multiple values.
- Elements are accessed using an index.
- Index starts from 0.
- Easy to traverse using loops.

---

## Time Complexity

| Operation | Complexity |
|-----------|------------|
| Access | O(1) |
| Search | O(n) |
| Insert (End) | O(1) Average |
| Delete | O(n) |

---

## Big O Notation

### O(1)

Constant Time

Example:

```python
print(numbers[2])
```

---

### O(n)

Linear Time

Example:

```python
for num in numbers:
    print(num)
```

---

### O(log n)

Logarithmic Time

Example:

Binary Search
