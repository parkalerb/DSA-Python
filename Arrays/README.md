# 📘 Arrays

## Problems Solved

### 1. Maximum Element in an Array

### Problem

Find the largest element in an array.

### Approach

Traverse the array and compare each element with the current maximum.

### Time Complexity

O(n)

### Space Complexity

O(1)

---

### 2. Reverse an Array

### Problem

Reverse all elements of an array.

### Approach

Used Python's built-in `reverse()` method.

### Time Complexity

O(n)

### Space Complexity

O(1)
